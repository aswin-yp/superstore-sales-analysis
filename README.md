# Superstore Sales Analysis & Business Insights

## Problem Statement
Analyze retail sales data to uncover trends in revenue, profitability, and regional performance to support business decision-making.

## Dataset
- Superstore sales dataset containing order, customer, product, and regional information

## Approach
- Cleaned data by handling missing values and removing duplicates
- Performed exploratory data analysis (EDA) to identify:
  - Sales trends over time (daily, monthly, yearly)
  - Top-selling and low-performing products
  - Regional sales and profitability patterns
- Conducted profit vs sales analysis
- Used visualizations to communicate insights effectively

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Key Insights
- Identified high-revenue but low-profit product categories
- Highlighted most and least profitable regions
- Observed seasonal sales trends impacting revenue

## Future Improvements
- Add forecasting models for sales prediction
- Build an interactive dashboard
- Segment customers for targeted analysis
